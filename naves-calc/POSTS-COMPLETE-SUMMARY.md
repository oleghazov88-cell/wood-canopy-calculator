## Реализованные типы столбов

| Тип | Описание | Основные характеристики |
|-----|----------|------------------------|
| **var-1** | Куст с подпятником | 4×80×80мм + перемычки + подпятник сверху |
| **var-2** | Колонна с вогнутостями | 270×270мм база + вогнутые плоскости + астрагал |
| **var-3** | Крупная колонна | 380×380мм база + 480×480мм астрагал |
| **var-4** | Компактная колонна | 280×280мм база + 300×300мм астрагал |
| **var-5** | Стандартный столб | Простое сечение + подпятник снизу |
| **var-6** | Композитная колонна | База + куст + капитель (самая сложная) |

## Структура куста столбов (var-1, var-6)

```
    ┌───────┬───────┐
    │ 80×80 │ 80×80 │
    ├───────┼───────┤  ← Перемычки каждые 510мм
    │ 80×80 │ 80×80 │
    └───────┴───────┘
    
    Расстояние между осями: 100мм
    Перемычки: 150×80мм
```

## Архитектурные столбы

### var-2: С вогнутостями
```
    ┌─────────┐
    │ 290×290 │ ← Астрагал
    ├─────────┤
    │╱       ╲│
    ││ 270×  ││ ← База с вогнутыми
    ││ 270   ││   плоскостями
    │╲       ╱│   (1035мм высота)
    └─────────┘
```

### var-6: Композитная
```
    ┌─────────┐
    │ 280×280 │ ← Капитель
    ├─────────┤
    │ 300×300 │ ← Астрагал верхний
    ├─────────┤
    │  КУСТ   │ ← 4 столба + перемычки
    │ 80×80×4 │
    ├─────────┤
    │ 300×300 │ ← Астрагал нижний
    ├─────────┤
    │ 280×280 │ ← База
    └─────────┘
```
