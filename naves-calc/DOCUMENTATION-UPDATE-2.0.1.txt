================================================================================
  💻 ПРИМЕРЫ КОДА
================================================================================

СТОЛБЫ (программно):
╔═══════════════════════════════════════════════════════════════╗
║ // Создаем геометрию программно (без файлов)                 ║
║ const geometry = new THREE.BoxGeometry(0.15, 3.0, 0.15);     ║
║ const material = new THREE.MeshStandardMaterial({...});      ║
║ const post = new THREE.Mesh(geometry, material);             ║
║ scene.add(post);                                              ║
║                                                               ║
║ // ✅ Нет загрузки файлов                                    ║
║ // ✅ Нет GLTFLoader                                         ║
╚═══════════════════════════════════════════════════════════════╝

РАСКОСЫ (импорт):
╔═══════════════════════════════════════════════════════════════╗
║ // Загружаем модель из файла                                 ║
║ const loader = new THREE.GLTFLoader();                       ║
║ const gltf = await loader.load('../raskos/r2.glb');         ║
║ const brace = gltf.scene;                                     ║
║ scene.add(brace);                                             ║
║                                                               ║
║ // ✅ Загрузка из GLB файла                                  ║
║ // ✅ Используется GLTFLoader                                ║
╚═══════════════════════════════════════════════════════════════╝

================================================================================
  🎓 ЧТО ТЕПЕРЬ ПОНЯТНО ИЗ ДОКУМЕНТАЦИИ
================================================================================

1. ✅ Столбы НЕ импортируются, а рисуются программно
2. ✅ Используется встроенный THREE.BoxGeometry
3. ✅ Нет зависимости от внешних файлов для столбов
4. ✅ Раскосы ИМПОРТИРУЮТСЯ из GLB файлов
5. ✅ Два разных метода для разных элементов
6. ✅ Причины выбора каждого метода
7. ✅ Преимущества программного рисования
8. ✅ Когда использовать импорт
