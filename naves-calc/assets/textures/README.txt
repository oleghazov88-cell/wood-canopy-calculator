ИНСТРУКЦИЯ ПО УСТАНОВКЕ ТЕКСТУРЫ
==================================

Для корректной работы перемычек в кусте столбов var-1 необходимо:

1. Поместите файл "Wood_Bamboo.jpg" в эту папку:
   naves-calc/assets/textures/

2. Текстура будет автоматически загружена при отображении столбов var-1

3. Если текстура не загрузится, проверьте:
   - Правильность имени файла (должно быть точно "Wood_Bamboo.jpg")
   - Наличие файла в этой папке
   - Консоль браузера на предмет ошибок (F12 -> Console)

Примечания:
- Текстура применяется только к перемычкам между столбами в кусте var-1
- Каждая перемычка получает уникальное смещение текстуры для разнообразия
- Столбы остаются с базовым деревянным материалом

Если у вас нет файла Wood_Bamboo.jpg, вы можете использовать любую другую 
текстуру дерева, переименовав её в Wood_Bamboo.jpg
